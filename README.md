# 🦊 Spooky Login Page

A fun, interactive login page built with **React**, **Axios**, and **React Router**.  
It features:

- Animated login **video feedback** (correct, wrong, again wrong)  
- **Flipping cards** for Success ✅ and Fail ❌ pages  
- Cool hover animations & smooth transitions  
- Fully responsive layout  

---

## 🎬 Demo

🔗👀 https://github.com/faaliha2007/login.git

---

## ⚡ Features

1. **Video Feedback**  
   - Shows different mp4 videos on login success or failure  
   - Plays `correct.mp4`, `wrong.mp4`, `againwrong.mp4` based on attempts  

2. **Animated Success Card**  
   - Flips on hover  
   - Green theme + confetti animation placeholder  
   - Logout button to return to login page  

3. **Animated Fail Card**  
   - Flips on hover  
   - Red theme + error message  
   - Try Again button to return to login page  

4. **Responsive Layout**  
   - Works on mobile and desktop  

---

## 💻 Technologies Used

- React 18  
- React Router DOM  
- Axios for backend login calls  
- CSS3 3D transforms for card flip  
- HTML5 Video
